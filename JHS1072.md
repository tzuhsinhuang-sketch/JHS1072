## 快速概覽

> 以好運龍寶 Super Buy 為基底，3x3 主盤面搭配右側 3x1 倍數輪。中心格（1,1）為獨立中心輪，可出現三種特殊 WILD，各自帶有飛散或倍數疊加效果，最強的 WW3 同時具備兩者。

| 項目 | 說明 |
|------|------|
| 專案基底 | 1052 好運龍寶 SuperBuy |
| 轉輪陣列 | 主盤面：3x3　＋　中心輪（位置 1,1）＋　倍數輪：3x1（右側）|
| 賠付線數 | 5 條固定賠付線 |
| 特殊功能 | 中心輪、特殊 WILD（WW1 / WW2 / WW3）、Extra Bet、Buy Feature × 2 |

### 符號一覽

| 類型 | 符號 | 出現位置 |
|------|------|---------|
| 一般符號 | M1、M2、M3、A、K、Q、J | 主盤面、中心輪 |
| WILD | WW | 主盤面、中心輪 |
| 特殊 WILD | WW1、WW2、WW3 | 中心輪限定 |
| 倍數符號 | 共 13 種，依模式出現不同組合（詳見核心機制 › 倍數輪）| 倍數輪 |

### 特殊功能摘要

| 功能 | 一句話說明 |
|------|-----------|
| 中心輪 | 獨立於主盤面的特殊轉輪，決定 WILD 效果種類 |
| WW1 | 飛散 WW 至主盤面後結算 |
| WW2 | 倍數輪三格相加後乘以盤面得分 |
| WW3 | 同時具備 WW1 與 WW2 效果，依序觸發 |
| Extra Bet | 加注 1.5X，調整倍數輪符號組合 |
| Buy Feature 1 | 100X 購買 8 次 Free Spins，WILD 機率提升 |
| Buy Feature 2 | 500X 購買 8 次 Free Spins，必出 WILD |

## 核心機制

### 基本遊戲

中獎規則：符號由最左至右連續出現方可獲獎。

### 中心輪

位於主盤面正中央（位置 1,1），為獨立轉輪，與其他轉輪分開滾動。

| 可出現符號 | 說明 |
|-----------|------|
| 一般符號 | 同主盤面一般符號 |
| WW | 一般 WILD，可替代所有一般符號 |
| WW1 / WW2 / WW3 | 特殊 WILD，僅出現於中心輪 |

### 特殊 WILD

所有特殊 WILD 除各自功能外，均具備一般 WILD 效果（可替代所有一般符號進行連線）。

強度順序：**WW3 > WW2 ≥ WW1 > 一般 WW**

| 符號 | 名稱 | 效果 |
|------|------|------|
| WW1 | 分散 WILD | 以中心輪為起點向外飛散，隨機將 1～8 個 WW 落至主盤面其餘 8 格，不覆蓋原有 WW，飛散完成後進行結算 |
| WW2 | 倍數 WILD | 倍數輪三格倍數相加後乘以主盤面得分（例：2X＋5X＋10X＝17X）|
| WW3 | 複合 WILD | 依序觸發 WW1 效果 → WW2 效果 |

Skip 規則詳見 [Skip](#Skip-功能) 章節。

### 倍數輪

| 項目 | 說明 |
|------|------|
| 陣列 | 3x1（同時顯示 3 個倍數）|
| 一般結算 | 主盤面有得分時，以第 2 行（中間格）倍數乘以盤面分數 |
| WW2 / WW3 觸發時 | 三格倍數全部相加（規則詳見[特殊 WILD › WW2](#特殊-WILD)）|

#### 倍數符號一覽（共 13 種）

| 模式 | 說明 |
|------|------|
| 總表 | 1X、2X、3X、5X、10X、15X、20X、25X、50X、100X、250X、500X、1000X |
| Normal | 1X、2X、3X、5X、10X、15X、20X、25X、50X、100X |
| Extra Bet | 移除 1X，增加 250X、500X、1000X |

### Extra Bet

| 項目 | 說明 |
|------|------|
| 加注倍率 | 1.5× Normal Bet |
| 效果 | 倍數輪符號變化詳見[倍數輪](#倍數輪)章節 |
| 開啟表演 | 開啟時有 ExtraBet 開啟表演提示玩家（飛出倍數符號到倍數輪） |
| 倍數輪提示 | 開啟時中獎框有對應視覺提示（外框或顏色變化）|
| BET欄位提示 | BET欄位有開啟提示（外框發光或有顏色外框） |

#### 按鈕

| 項目 | 說明 |
|------|------|
| 按鈕文案 | 僅顯示 EXTRABET，不顯示加注金額（文案參考 Languages）|
| 按鈕行為 | 點擊切換 Extra Bet 開關，按鈕具開關示意 |
| 操作限制 | 僅 Idle 狀態可操作 |

### Buy Feature

兩種 Buy Feature，共用同一個購買確認視窗，視窗內提供兩個選項供玩家選擇。

| | BF1 | BF2 |
|---|---|---|
| 價格 | 100× Bet | 500× Bet |
| 場次 | 8 次 Free Spins | 8 次 Free Spins |
| 特殊效果 | WILD（一般 ＋ 特殊）出現機率提升 | 每次必出一般 WW 或特殊 WILD |
| 倍數輪 | 最低 2X，最大 1000X | 最低 2X，最大 1000X |
| 流程規則 | 同 Base Game | 同 Base Game |

#### 入口按鈕

| 項目 | 說明 |
|------|------|
| 按鈕文案 | 僅顯示 BUY FEATURE，不顯示金額（文案參考 Languages）|
| 按鈕行為 | 點擊開啟購買確認視窗 |


#### 購買確認視窗

```wf
title: BUY FEATURE
option: 說明文案 | 100.00 | selected
option: 說明文案 | 500.00
bet: 1.00
action: 取消 | secondary
action: BUY | primary
```

| 元素 | 說明 |
|------|------|
| 選項 | 同時顯示 BF1 / BF2 兩個選項供選擇 |
| 規格說明文案 | 說明各選項的 WILD 機率提升、倍數輪條件等（文案參考 Languages）|
| 顯示購買金額 | 依當前 Bet 換算後顯示 |
| Bet 調整欄位 | 可調整 Bet 金額 |
| 購買確認按鈕 | 確認購買所選選項 |
| 關閉按鈕 | 關閉視窗，Bet 不變 |

## 滾輪行為

SPIN 時，所有轉輪同時轉動，停輪則如下順序停輪。

### 停輪順序

```
R1 → R2 → R3 → 中心輪 → 倍數輪
```

> 一般轉輪停輪期間，中心輪仍持續滾動，讓玩家可同時看到中心輪轉動→停輪的過程，接著停倍數輪。

### 整體流程

```
IDLE → SPIN → 3x3 依序停輪（中心輪除外）→ 中心輪停輪 → 倍數輪停輪 → 依中心輪結果進行結算
```

## 贏分秀線

演出順序：先秀線，再出現贏分看板

| 項目 | 說明 | 顯示內容 |
|------|------|---------|
| Show All Win | 同時秀出所有得分線 | 得分線條、線號<br>得分符號表演<br>未連線獎項壓黑，倍數輪不壓黑 |
| Loop Win | 依得分線編號依序單條顯示，loop 完所有得分線後回到 Show All Win，再次進入 Loop Win 循環 | 同上 |

### 觸發時機

| 時機 | 行為 |
|------|------|
| 贏分演出 | Show All Win |
| 結算後 | Show All Win → Loop Win 循環 |

## 贏分看板

贏分看板的出現與收起時機詳見遊戲流程圖。

### 看板類型

| 類型 | 觸發情境 | 說明 |
|------|---------|------|
| 純分數版 | 一般 / WW、WW1 | 顯示單一得分數字 |
| 倍數版 | WW2、WW3 | 分前後兩段：前段顯示盤面贏分，後段空白供倍數輪倍數收集，收集完成後兩數相乘顯示總分 |

### 動態尺寸

| 項目 | 說明 |
|------|------|
| 尺寸調整 | 看板寬度隨分數位數動態調整 |
| 最大寬度 | 不超過 3×3 主盤面寬度範圍 |
| 超出處理 | 縮小分數字體大小，看板外框維持不變 |

## Winframe

Winframe 具有兩種顯示模式：跑馬燈與贏分資訊。

### 跑馬燈

顯示遊戲特色介紹等循環文案。文案內容詳見文案。

**文字捲動規則**

| 情境 | 行為 |
|------|------|
| 文字長度在範圍內 | 靜態顯示 1500ms → 切換下一句 |
| 文字長度超出範圍 | 出現後靜置 1500ms → 往左捲動直至完整顯示 → 停止 1500ms → 切換下一句 |
| 最後一句結束後 | 循環回第一句 |

### 贏分資訊

贏分結算後，於得分線 Loop 期間顯示各得分線分數。顯示格式詳見文案。

## 大獎表演

升階式表演，從 BIG WIN 開始，依最終得分倍數決定換階時機與最高階層。得分倍數以 Normal Bet 數值計算。

### 觸發門檻

| 階層 | 觸發條件（Normal Bet 倍數）|
|------|--------------------------|
| BIG WIN | ≥ 5x |
| MEGA WIN | ≥ 15x |
| SUPER WIN | ≥ 30x |

### 升階規則

| 達成階層 | 換階時機 |
|---------|---------|
| BIG WIN | 固定表演時長，不換階 |
| MEGA WIN | 滾分至總得分 1/2 時，BIG WIN → MEGA WIN |
| SUPER WIN | 滾分至總得分 1/3 時，BIG WIN → MEGA WIN；2/3 時，MEGA WIN → SUPER WIN |

Skip 規則詳見 [Skip](#Skip-功能) 章節。

## 二段加速

**預設狀態**：Normal

| 模式 | 效果 | 備註 |
|------|------|------|
| Normal | 標準速度，Spin 時各輪同時滾動<br>停輪時依照停輪順序<br>有瞇牌 | — |
| Quick Mode | 各輪同時滾動、同時停輪<br>無瞇牌<br>特殊 WW 的表演不加速 | 特殊 WW 飛散、倍數等演出仍以正常速度播放，其餘轉輪行為維持 Quick 速度 |
| Turbo Mode | 基於 Quick Mode，整體加速 1.8x（暫定）| — |

## Auto Play

| 項目 | 說明 |
|------|------|
| 次數 | 50、100、200、500、999、∞ |
| 二段加速 | Auto Play 進行中仍可調整 |

## 瞇牌表演

| 觸發條件 | 說明 |
|---------|------|
| 前三輪盤面贏分 ≥ 5X | 倍數輪進行瞇牌表演 |
| 中心輪停輪為 WW2 或 WW3 | 倍數輪進行瞇牌表演 |
| Quick / Turbo 加速模式 | 無瞇牌表演 |

Skip 規則詳見 [Skip](#Skip-功能) 章節。

## Skip 功能

玩家可點擊畫面任意位置或 Spin 按鈕跳過表演，直接顯示最終結果。Auto 模式下亦可操作。

| 表演項目 | 可否 Skip | 觸發後延遲 | Skip 後結果 |
|---------|----------|-----------|------------|
| 瞇牌表演 | 可 | 300ms | 表演收起，完全停輪 |
| 大獎表演 | 可 | 300ms | 跳至最終贏分顯示 |
| WW1 分散 WILD | 可 | 300ms | 跳至分散完成結果，接著演出連線中獎或無中獎結束 |
| WW2 倍數輪中獎框擴展 | 不可 | — | — |
| WW2 收集倍數 | 可 | 300ms | 跳至贏分看板顯示最終收集倍數，接著演出分數相乘 |
| WW3 分散 WILD ＋ 倍數輪中獎框擴展 | 可 | 300ms | 跳至分散完成結果 ＋ 倍數輪中獎框完成擴展，接著演出連線中獎或無中獎結束 |
| WW3 收集倍數 | 可 | 300ms | 跳至贏分看板顯示最終收集倍數，接著演出分數相乘 |

## 預設盤面

開啟 Extra Bet 時，盤面換成 Extra Bet 預設盤面；關閉時盤面不變動，倍數輪切回 Normal Bet 預設，並關閉 Extra Bet 開啟的提示。

### Normal Bet

```board
R1 | R2 | R3 | R4
M1 | WW | M2 | 20x
M1 | WW3 | M2 | 100x
M1 | WW | M2 | 15x
```

### Extra Bet

```board
R1 | R2 | R3 | R4
M1 | WW | M2 | 500x
M1 | WW3 | M2 | 1000x
M1 | WW | M2 | 250x
```
